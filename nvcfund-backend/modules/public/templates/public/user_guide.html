Troubleshooting

Common Issues
  • Login Problems: Clear browser cache and cookies
  • Transfer Delays: Check network status and transaction limits
  • KYC Verification: Ensure document quality and expiry dates
  • Service Errors: Verify Service keys and endpoint URLs
Quick Fixes
  • Page Loading Issues: Refresh page or try incognito mode
  • Mobile App: Update to latest version from app store
  • 2FA Problems: Sync device time or use backup codes
  • Account Locked: Contact support for immediate assistance
