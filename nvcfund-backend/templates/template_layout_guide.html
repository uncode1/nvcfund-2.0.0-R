    Responsive Design

    Mobile-First
    Breakpoints
    Device Breakpoint Layout
    Mobile < 768px Single column
    Tablet 768px - 1200px Stacked layout
    Desktop > 1200px Multi-column
    Layout Adaptation
    • Mobile: Sidebar moves to top
    • Tablet: Reduced spacing
    • Desktop: Full layout with sidebar
    Note: All layouts automatically adapt to screen size using CSS Grid and Flexbox.

    Best Practices

    Important
    ✅ Do's
    • Use semantic HTML structure
    • Implement proper ARIA labels
    • Follow consistent spacing
    • Use unified design system classes
    • Include breadcrumb navigation
    • Organize content in logical sections
    ❌ Don'ts
    • Don't use inline styles
    • Don't create standalone HTML pages
    • Don't ignore responsive design
    • Don't hardcode navigation links
    • Don't skip accessibility features
    • Don't use inconsistent spacing
