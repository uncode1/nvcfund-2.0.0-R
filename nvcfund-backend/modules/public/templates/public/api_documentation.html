Service Overview

The NVC Banking Platform provides a comprehensive RESTful Service for integration with your applications. Our Service enables secure account management, transaction processing, and real-time banking operations.

Base URL
https://api.nvcbanking.com/v1/
Authentication

Service Key required in header: X-Service-Key

Rate Limits
  • Standard: 1000 requests/hour
  • Premium: 10,000 requests/hour
  • Enterprise: Unlimited

Authentication

Service Key Generation

Generate Service keys from your account dashboard under Service Management. Each key has specific permissions and rate limits.

Example Request Header
X-Service-Key: your-api-key-here
Content-Type: application/json

Account Endpoints

Method Endpoint Description
GET /accounts List all accounts
GET /accounts/{id} Get account details
POST /accounts Create new account
PUT /accounts/{id} Update account details

Transaction Endpoints

Method Endpoint Description
GET /transactions List transactions
GET /transactions/{id} Get transaction details
POST /transfers Initiate transfer
POST /payments Process payment

NVCT Stablecoin Endpoints

Method Endpoint Description
GET /nvct/balance Get NVCT balance
POST /nvct/transfer Transfer NVCT tokens
GET /nvct/rate Get current USD rate
POST /nvct/exchange Exchange USD to NVCT
